<div>
  <details>
<summary><strong>📝 ملاحظة حول TypeScript</strong></summary>
<br>
رغم أن الكود النهائي مكتوب بـ <strong>HTML</strong> و <strong>CSS</strong> و <strong>JavaScript</strong> فقط،
إلا أن <strong>TypeScript</strong> يُستعمل كمساعد أثناء التطوير لتحسين منطق الكود وتنظيمه،
لكن لا يتم تضمينه في الكود النهائي داخل المشاريع.
    <br><br>
لكن لا تخف إذا لم تكن متعلمًا TypeScript بعد، فيمكنك إنشاء المشاريع بكل سهولة إذا كنت متأسس في JavaScript.
</details>
</div>
